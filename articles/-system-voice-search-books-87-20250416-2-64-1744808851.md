# 番薯系统支持语音搜索图书吗？

### **4. 借书和还书管理**
无论是自助借还还是管理员代为操作，系统提供灵活的借还方式，满足不同书馆的运营需求。在借书过程中，系统记录每一本书籍的信息和借阅者，确保可追溯性。
